# Beispielkonfigurationen

Diese Dateien sind sichere Vorlagen fuer eine lokale Installation.

Vorgehen:

1. Benoetigte `.example.txt`-Datei nach `_config/` kopieren.
2. `.example` aus dem Dateinamen entfernen.
3. Eigene lokalen Werte eintragen.
4. Echte Zugangsdaten, E-Mail-Adressen, Kundendaten und private Pfade niemals committen.

Hinweise:

- `_config/` ist fuer private Werte vorgesehen und wird ignoriert.
- Passwoerter und SMTP-/Outlook-Zugangsdaten gehoeren nicht ins Repository.
- Fuer ein oeffentliches Repository nur diese Beispielkonfigurationen verwenden.
