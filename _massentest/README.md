# Massentest

Massentest-Referenzen und Ergebnisse werden lokal erzeugt.

Wichtig:

- `_massentest/ergebnisse/` enthaelt echte Sollwerte, Istwerte, Statusdaten und Snapshots.
- Ergebnisse und Snapshots nicht veroeffentlichen.
- Referenzdokumente nur lokal verwenden oder durch anonymisierte Beispieldaten ersetzen.
- Fuer ein Public-Repository keine echten Dokumentnamen, OCR-Texte oder Kundendaten committen.
